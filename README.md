# Barinfrick compiler
Brainfuck compiler, but made really fast and with spelling mistakes

## Usage
### Prerequisites
Just some basic stuff:
- Modern C/C++ compiler
- CMake installed

### Building the project
You can download a zip file or use git:
```bash
> git clone \
    https://github.com/Reivaxmar/Barinfrick-compiler \
    Barinfrick-compiler
> cd Barinfrick-compiler
```

Great! Now you have the project, now you just need to build it:
```bash
> bash ./build.sh
> cmake --build .
> ./barinfrick
Hello, world!
```

Now your project should be up and running.
